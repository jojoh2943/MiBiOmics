# MiBiOmics
MiBiOmics is a shiny-based web application to perform correlation network analysis and multi-omics analysis on omics datasets

## Installation

MiBiOmics is available at https://shiny-bird.univ-nantes.fr/jzoppi/app/ but can also be runned locally in RStudio. To run MiBiOmics locally you need to install the lastest version of shiny, **RStudio (version 3.5)** and run the following commands:

```r
library(shiny)
runGitHub("MiBiOmics", "jojoh2943")
```
All the necessary packages will be installed automatically for the usage of MiBiOmics.
